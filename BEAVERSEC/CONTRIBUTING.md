# Guia de Contribuição - BeaverSec

Obrigado por considerar contribuir com o **BeaverSec**!

## 🐛 Reportando Bugs
Abra uma [issue](https://github.com/ojhonatanls/beaversec/issues) com:
1. Título claro
2. Passos para reproduzir
3. Comportamento esperado vs atual
4. Ambiente (SO, Python, versão)

## 🚀 Enviando Código (Pull Requests)
1. Fork o repositório
2. Clone: `git clone https://github.com/seu-usuario/beaversec.git`
3. Crie uma branch: `git checkout -b feature/nova-feature`
4. Faça suas alterações
5. Teste: `python main.py -l`
6. Commit: `git commit -m "➕ Adiciona funcionalidade X"`
7. Push: `git push origin feature/nova-feature`
8. Abra um Pull Request

## 📝 Padrões de Código
- 4 espaços para indentação
- Docstrings no formato Google/NumPy
- Type hints para parâmetros e retornos
- Linhas com no máximo 100 caracteres

**Agradecemos sua contribuição!** 🦫🚀
